# Resume Builder

## Setup

1. Ensure Node.js is available via the internal Antigravity runner.
2. Open a terminal in the `resume-builder` directory.
3. Start the server:
   ```
   agy-node.cmd server.js
   ```

## Usage

- Open your browser and navigate to `http://localhost:3000`.
- Use the controls on the left to edit resume details.
- Click **Optimize with AI** to mock AI enhancement.
- Click **Upgrade (Stripe Mock)** to simulate a subscription.

## Development

- Edit `index.html` to modify the UI.
- The server serves any static assets placed in this folder.

Enjoy building your résumé!
